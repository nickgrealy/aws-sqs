Имя или URL-адрес очереди сообщений, существующей в службе Amazon Simple Queue Service (SQS). Плагин будет запускать сборку, когда сообщение опубликовано в этой очереди.

Очередь не будет активно отслеживаться до тех пор, пока не будет настроено хотя бы одно задание для прослушивания сообщений из этой очереди. Смотрите конфигурации триггера сборки отдельных задач.

Дополнительная документация доступна по адресу AWS SQS Plugin на GitHub.
